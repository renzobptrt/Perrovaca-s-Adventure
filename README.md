# The Last Knight
Rogue-like rpg teniendo como personaje principal a un conejo
Corre en version de Unity 2019.3.0b6

## Characteristics
- Type: 2D
- Point of view: Third person
- Gender: RPG
- Platform: PC / Mobile
- Graphics engine: Unity 2D
- Main language: C#
- Level : 1
